# Void Gambler Electron + Discord RPC

## Setup
1. Install Node.js 18+.
2. From this folder, run:
   npm install
3. Start desktop app:
   npm start

## Notes
- Discord desktop client must be running for Rich Presence to appear.
- RPC client/application ID is set to: `1482793632263049389`.
- Presence updates are sent from the game scenes (menu, in-run, boss fights, game over).
